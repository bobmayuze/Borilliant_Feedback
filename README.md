# Borilliant Feedback
This is a repo built for better feedback of courses at rpi.

## Install 
```
$ npm install 
```

## Running Instructions:

Open mongo DB:

Step 1:
Setting the dbpath of mongodb to the path I want
```
sudo mongod --dbpath data/db/
```

Step 2:
```
mongo
```

Start mongodb

Step 3:
```
nodemon
```
